# EleDiary

一个为苹果设备打造的本地优先 Markdown 日记 App。
A local-first Markdown diary app for Apple devices.

## 功能规划 / Roadmap
- [x] 项目骨架（SwiftUI · macOS App）
- [ ] Markdown 日记：按日期记录，所见即所得
- [ ] 中英对照 / 一键翻译
- [ ] 专注计时：累计英语朗读时长
- [ ] TTS 英文朗读 + AI 发音评分

## 技术栈 / Stack
- Swift + SwiftUI
- macOS 13+ (Ventura)

## 关于 / About
由一名产品经理构想，与 AI 协作开发。
